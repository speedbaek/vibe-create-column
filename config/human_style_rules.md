# 사람냄새 글쓰기 규칙

## 문장 구조 다양성
- 한 문단 안에서 같은 종결어미를 3회 이상 반복하지 말 것 (~습니다, ~습니다, ~습니다 금지)
- 문장 길이를 의도적으로 섞을 것: 짧은 문장(10자 이하) + 중간(20~40자) + 긴 문장(50자 이상) 교차
- 때때로 체언 종결(명사로 끝나는 문장)을 섞을 것. 예: "이것이 핵심입니다." → "이것이 핵심."

## 구어체 혼합
- 전체적으로 ~습니다/~합니다 경어체를 기본으로 하되, 중간중간 구어체를 자연스럽게 섞을 것
- 허용되는 구어체 예시: "그거 아시나요?", "솔직히 말하면", "뭐 그런 거죠", "한마디로"
- 독자에게 직접 말을 거는 듯한 표현 사용: "여러분", "혹시", "~하신 분들", "생각해 보셨나요?"

## 감정 표현
- 가끔 감탄사를 넣을 것: "참", "글쎄요", "사실", "솔직히"
- 강조할 때 "정말", "진짜", "절대" 등 구어적 강조 사용
- 과도한 감정 표현은 피하되, 완전히 무감정한 톤도 피할 것

## 불완전성
- 완벽하게 정리된 글보다 약간의 사족이나 여담이 있는 편이 자연스러움
- 가끔 "(이 부분은 추후 별도 컬럼으로 설명 드릴 예정입니다)" 같은 부가 언급 추가
- 주제에서 살짝 벗어났다 돌아오는 패턴: "잠깐 딴 이야기를 했는데" "다시 본론으로 돌아가면"

## 단락 구분
- 네이버 블로그 특성상 빈 줄(​)을 활용한 여백 있는 단락 구분
- 한 단락에 3~5문장 정도가 적절
- 중요 포인트 전에 한 줄 비워서 강조 효과

## 금지 사항
- "~것은 부인할 수 없습니다", "~라고 할 수 있겠습니다" 같은 논문투 금지
- "다양한", "중요한", "효과적인" 같은 형용사 남발 금지
- 영어 단어를 굳이 섞지 말 것 (한국어로 충분한 경우)
- "첫째, 둘째, 셋째" 같은 기계적 나열보다 자연스러운 흐름 선호
