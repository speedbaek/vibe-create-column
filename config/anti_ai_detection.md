# AI 탐지 방지 규칙

## 문장 패턴 회피
- "~의 중요성을 강조하고 싶습니다" → 직접적으로 설명할 것
- "~에 대해 알아보겠습니다" → 바로 내용으로 들어갈 것
- "결론적으로 말씀드리면" → "정리하면" 또는 아예 쓰지 말 것
- "~하는 것이 바람직합니다" → "~하세요" 또는 "~해야 합니다"로 직설적으로

## 구조 패턴 회피
- 서론-본론-결론의 교과서적 3단 구조를 그대로 따르지 말 것
- 모든 섹션을 동일한 길이로 쓰지 말 것 (어떤 섹션은 길고, 어떤 것은 짧게)
- 각 섹션 시작을 "먼저~", "다음으로~", "마지막으로~" 패턴으로 시작하지 말 것
- 하나의 글에서 리스트(불릿)를 3번 이상 사용하지 말 것

## 어휘 패턴 회피
- 금지 단어: "핵심적", "효과적", "체계적", "종합적", "전략적", "필수적"
- 금지 구문: "~의 관점에서", "~를 고려할 때", "~라는 점에서", "~에 기반하여"
- 금지 접속사 남발: "또한", "더불어", "아울러", "나아가" (1개 이내 사용)
- "다양한"은 글 전체에서 1회 이하로 제한

## 자연스러움을 위한 기법
- 질문을 던지고 스스로 답하는 자문자답 패턴 활용
- 실제 상담 경험담을 자연스럽게 녹여낼 것 ("최근 상담한 고객 중에~")
- 비유/은유 사용: "러시안 룰렛 같은", "부메랑이 되어 돌아옵니다"
- 숫자와 구체적 사례를 통한 신뢰감 부여

## 반복 방지
- 같은 단어를 한 문단 내에서 2회 이상 사용하지 말 것
- 문장 시작을 같은 패턴으로 3번 연속 하지 말 것
- 접속사를 빠르게 교체: "그런데" → "한편" → "다만" → "사실"
