# AI 탐지 방지 규칙

## 절대 금지 표현 목록

아래 표현이 글에 포함되면 즉시 "AI가 쓴 글"로 인식됩니다.
하나라도 사용하면 실패입니다.

### 금지 단어/표현
- "포괄적으로", "종합적으로", "체계적으로"
- "살펴보겠습니다", "알아보겠습니다", "정리해 보겠습니다"
- "마무리하겠습니다", "마무리하며"
- "~에 대해 알아보겠습니다" (서론 시작 패턴)
- "오늘은 ~에 대해 이야기해 보겠습니다"
- "지금부터 ~를 살펴보도록 하겠습니다"
- "결론적으로 말씀드리자면"
- "그럼 하나씩 살펴볼까요?"

### 금지 구조 패턴
- 모든 소제목 아래 내용 길이가 거의 같은 구조
- "첫째, 둘째, 셋째"로 완벽하게 나열되는 병렬 구조
- 모든 문단이 비슷한 길이로 균일한 구조
- 서론-본론-결론이 교과서처럼 완벽하게 분리된 구조

### 금지 형용사 남발
- "다양한", "중요한", "핵심적인", "효과적인", "효율적인"
- 이런 형용사를 글 전체에서 2회 이상 쓰면 안 됩니다.
- 대신 구체적으로 뭐가 다양하고, 왜 중요한지를 풀어서 쓰세요.

## 불균형 원칙

### 소제목별 내용 길이
- 첫 번째 소제목: 3~5문장
- 두 번째 소제목: 7~10문장 (가장 길게)
- 세 번째 소제목: 2~4문장 (짧게)
- 이런 식으로 의도적으로 불균형하게 작성하세요.

### 문장 길이 변화
- 긴 문장(40자) → 짧은 문장(15자) → 중간 문장(25자)
- 리듬감 있게 길이를 섞으세요.
- 특히 짧은 단독 문장이 강조 효과가 있습니다.

### 전환 표현
- 매끄러운 전환보다 약간 거친 전환이 자연스럽습니다.
- "한편,", "그런데,", "참고로," 같은 짧은 전환어 사용
- 가끔 전환어 없이 바로 다음 주제로 넘어가기도 하세요.

## 어미 반복 방지

- 연속 3문장이 같은 어미(~습니다)로 끝나면 안 됩니다.
- ~습니다 → ~는데요 → ~습니다 → ~겠지요 식으로 섞으세요.
- 특히 ~입니다/~습니다가 4줄 이상 연속되면 AI 느낌이 강해집니다.

## 감정 곡선

글 전체에 감정의 흐름이 있어야 합니다:
1. 시작: 담담하게 (인사, 상황 설명)
2. 중반: 공감/걱정 (독자의 고민에 공감, 사례 인용)
3. 절정: 단호함/확신 ("이건 반드시 ~해야 합니다")
4. 마무리: 따뜻함 ("여러분의 성공을 기원합니다")

이 곡선이 없이 처음부터 끝까지 같은 톤이면 기계가 쓴 글입니다.
