# 블로그 제목 스타일 가이드

## 핵심 원칙
- 키워드는 **제목 앞부분**에 배치 (SEO 최적화)
- AI가 쓴 느낌 절대 금지. 실제 전문가가 직접 쓴 느낌 필수
- 과도한 이모지, 감탄사, "꿀팁", "필독" 같은 인터넷 클리셰 금지
- **매번 다른 패턴**을 사용할 것 (같은 패턴 연속 사용 절대 금지)

## 제목 패턴 (14가지) — 반드시 서로 다른 패턴 사용

### 1. 쉼표 패턴
키워드 + 쉼표 + 후킹 문구
- 특허청상표등록, 변리사를 통해 해야 하는 이유 3가지
- 상표등록비용, 4만원짜리의 무서운 진실
- 디자인등록출원, 이 3가지 실수 하면 99% 거절됩니다

### 2. 질문형
독자가 실제로 궁금해할 만한 질문 형태
- 아이디어특허등록, 시제품 없어도 신청 가능한가요?
- 상호등록하려는데요 상표랑 다른 점이 뭔가요?

### 3. 부정/경고형
손실 회피 심리 자극
- BM 특허등록 전문 변리사, 잘 못 찾으시면 개고생 합니다.
- 특허내는방법, 망하는 사람들의 전형적인 특징

### 4. {feat.} 부연 패턴
본문 핵심 정보를 중괄호로 부가 설명
- 긴급특허{feat. 임시명세서 활용한 특허가출원}
- 상표출원 주의사항{feat. 셀프 상표등록의 위험성}

### 5. "이것" 강조 패턴
핵심 포인트를 '이것'으로 감춰서 클릭 유도
- 특허변리사 추천, 비용보다 중요한 '이것' 기억하세요

### 6. 구어체/솔직한 톤
전문가 본인이 직접 말하는 느낌
- 특허소송, 변리사 VS 변호사 딱 까놓고 말할께요
- 변리사가 알려주는 상표등록, 읽지 마세요 (진짜 비법이라)

### 7. 경력 어필형
전문성과 신뢰를 제목에서 바로 보여주기
- 상표등록절차, 14년 경력 변리사가 직접 쓰는 글

### 8. VS 비교형
두 가지를 대비시켜 궁금증 유발
- 셀프 상표등록 VS 변리사 대리, 뭐가 나을까?

### 9. 독자 대화체
독자가 직접 질문하는 듯한 구어체
- 특허등록 하고 싶은데, 뭐부터 해야 하나요?

### 10. 괄호 강조형
꺾쇠/소괄호로 타겟 독자나 핵심 포인트를 강조
- 특허출원비용, 진행 전 착각하는 3가지 이유<중소기업 대표 필독>
- 상표등록 절차, 초보자가 놓치는 함정 (대표님 주의)
- 디자인권 침해 대응법<스타트업 CEO가 꼭 알아야 할 것>
- 실용신안 vs 특허, 어떤 게 유리할까 (비용 비교 포함)

### 11. 정보성 짧은 제목
검색 의도에 바로 답하는 심플하고 깔끔한 제목 (30자 이하 OK)
- 상표등록 절차 및 비용 핵심만 정리
- 특허출원 방법, 준비서류부터 등록까지
- 디자인등록 비용과 소요기간 총정리
- 실용신안 등록 요건 및 절차 안내

### 12. 숫자 리스트형
구체적 숫자로 정보량을 어필 (N가지, TOP N)
- 상표등록 전 확인해야 할 5가지 체크리스트
- 특허 거절 사유 TOP 3, 이것만 피하면 됩니다
- 변리사가 추천하는 IP 보호 전략 4가지
- 디자인권 등록 성공률 높이는 방법 3선

### 13. 비용 임팩트형
구체적 금액으로 손실/이익을 체감시키는 패턴
- 상표출원비용, 4만원 아끼려다 4천만원 날립니다
- 특허출원 관납료 줄이는 법, 최대 70% 감면 가능
- 상표등록비용 20만원 vs 200만원, 차이가 뭘까요
- 디자인등록 40만원이면 끝? 숨겨진 비용까지 알려드립니다

### 14. 고객 사례/스토리형
실제 상담 경험을 녹여 공감 유발
- 3년 전 특허 안 냈으면 지금 사업 접었습니다
- "상표 빼앗겼어요" 상담 후 30분 만에 해결한 방법
- 대표님, 그 상표 진짜 괜찮은 거 맞습니까

## 제목 생성 규칙

### 반드시 지켜야 할 것
1. 키워드를 제목 앞부분에 자연스럽게 배치
2. **14가지 패턴 중 매번 서로 다른 패턴** 사용 (동일 패턴 반복 절대 금지)
3. 30~50자 사이 길이 (정보성 짧은 제목은 25자도 허용)
4. 실제 사람이 쓴 것 같은 자연스러운 어투
5. 독자(사업자, 창업자, 발명가)의 실제 고민을 반영

### 절대 하면 안 되는 것
1. "완벽 가이드", "궁극의", "올인원" 같은 AI스러운 표현
2. "~의 모든 것", "~A to Z" (너무 흔한 패턴)
3. 영어 단어 남발 (한국어 자연스러움 유지)
4. 이모지를 제목에 넣기
5. "꿀팁", "필독", "강추" 같은 인터넷 클리셰
6. 지나치게 긴 제목 (50자 초과)
7. 같은 패턴만 반복 사용
