# 윤웅채 변리사 블로그 제목 스타일 가이드

## 핵심 원칙
- 키워드는 **제목 앞부분**에 배치 (SEO 최적화)
- 키워드 뒤에 **쉼표(,)**로 구분한 후 후킹 문구 추가
- AI가 쓴 느낌 절대 금지. 실제 변리사가 직접 쓴 느낌 필수
- 과도한 이모지, 감탄사, "꿀팁", "필독" 같은 인터넷 클리셰 금지

## 제목 패턴 (10가지)

### 1. 쉼표 패턴 (가장 빈번)
키워드 + 쉼표 + 후킹 문구
- 특허청상표등록, 변리사를 통해 해야 하는 이유 3가지
- 상표등록비용, 4만원짜리의 무서운 진실
- 특허출원비용, 저렴한 곳만 찾다가 큰 코 다칩니다
- 디자인등록출원, 이 3가지 실수 하면 99% 거절됩니다

### 2. 질문형
독자가 실제로 궁금해할 만한 질문 형태
- 아이디어특허등록, 시제품 없어도 신청 가능한가요?
- 특허출원등록, 진행시 개인 or 법인 명의 뭐가 좋을까요?
- 상호등록하려는데요 상표랑 다른 점이 뭔가요?

### 3. 숫자/통계형
구체적 수치로 신뢰감과 호기심 자극
- 특허청 디자인등록출원, 성공율 98%의 비밀
- 특허비용 대략 300만원, 포함 내역 총정리
- 상표등록 거절율 TOP 5 사유, 미리 알면 피할 수 있습니다

### 4. 부정/경고형
손실 회피 심리 자극
- BM 특허등록 전문 변리사, 잘 못 찾으시면 개고생 합니다.
- 특허내는방법, 망하는 사람들의 전형적인 특징
- 특허출원비용, 저렴한 곳만 찾다가 큰 코 다칩니다

### 5. {feat.} 부연 패턴
본문 핵심 정보를 중괄호로 부가 설명
- 긴급특허{feat. 임시명세서 활용한 특허가출원}
- 상표출원 주의사항{feat. 셀프 상표등록의 위험성}

### 6. "이것" 강조 패턴
핵심 포인트를 '이것'으로 감춰서 클릭 유도
- 특허변리사 추천, 비용보다 중요한 '이것' 기억하세요
- 상표권침해, 대응 전에 반드시 확인해야 할 '이것'

### 7. 구어체/솔직한 톤
변리사 본인이 직접 말하는 느낌
- 특허소송, 변리사 VS 변호사 딱 까놓고 말할께요
- 특허출원절차, 솔직히 말하면 이것만 알면 됩니다
- 변리사가 알려주는 상표등록, 읽지 마세요 (진짜 비법이라)

### 8. 경력 어필형
전문성과 신뢰를 제목에서 바로 보여주기
- 상표등록절차, 14년 경력 변리사가 직접 쓰는 글
- 발명특허 출원변리사 비용, 최저가 줄세우기?

### 9. VS 비교형
두 가지를 대비시켜 궁금증 유발
- 레시피 음식특허 등록 VS 영업비밀 보호
- 특허소송, 변리사 VS 변호사
- 셀프 상표등록 VS 변리사 대리, 뭐가 나을까?

### 10. 독자 대화체
독자가 직접 질문하는 듯한 구어체
- 상호등록하려는데요 상표랑 다른 점이 뭔가요?
- 특허등록 하고 싶은데, 뭐부터 해야 하나요?

## 제목 생성 규칙

### 반드시 지켜야 할 것
1. 키워드를 제목 앞부분에 자연스럽게 배치
2. 위 10가지 패턴 중 하나를 활용 (매번 다른 패턴 권장)
3. 30~50자 사이 길이
4. 실제 사람이 쓴 것 같은 자연스러운 어투
5. 독자(사업자, 창업자, 발명가)의 실제 고민을 반영

### 절대 하면 안 되는 것
1. "완벽 가이드", "궁극의", "올인원" 같은 AI스러운 표현
2. "~의 모든 것", "~A to Z", "~총정리" (너무 흔한 패턴)
3. 영어 단어 남발 (한국어 자연스러움 유지)
4. 이모지를 제목에 넣기
5. "꿀팁", "필독", "강추" 같은 인터넷 클리셰
6. 지나치게 긴 제목 (50자 초과)
7. "~해야 하는 N가지 이유"만 반복 (다양한 패턴 활용)
